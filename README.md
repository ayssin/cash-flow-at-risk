\# Cash Flow at Risk (CFaR)



\## Описание проекта

Расчёт риска денежных потоков банка тремя методами на основе реальных транзакций (2015-2019).



\## Результаты (95% доверительный уровень)

|-------|------|

| Исторический | -75,056,454.40 |

| Параметрический | -94,513,993.20 |

| Монте-Карло | -5,313,078,368.19 |
