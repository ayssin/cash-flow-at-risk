# Cash Flow at Risk (CFaR)

Посчитали риск денежных потоков банка на основе 1,294 дней транзакций (2015-2019).

## Результаты (95% доверительный уровень)

| Метод | CFaR |
| ----- | ---- |
| Исторический | -75,056,454 |
| Параметрический | -94,513,993 |
| Монте-Карло | -5,313,078,368 |

Вывод: В 95% случаев дневные потери не превышают указанную сумму. Метод Монте-Карло даёт более консервативную оценку.

## Данные

- Период: 01.01.2015 – 05.03.2019
- Дней в выборке: 1,294
- Суммарный приток: 238.5 млрд
- Суммарный отток: 240.4 млрд
- Чистый поток: -1.9 млрд
