\# Cash Flow at Risk (CFaR)



\## Описание проекта

Расчёт риска денежных потоков банка тремя методами.



\## Методология

\- \*\*Исторический VaR\*\* — перцентиль эмпирического распределения

\- \*\*Параметрический VaR\*\* — с предположением нормальности

\- \*\*Монте-Карло\*\* — бутстрап исторических дневных потоков
